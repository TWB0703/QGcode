# **数据容器**
## **列表**
1. 列表的定义方式及特点：
* 列表名 = [元素1 ， 元素2 ，元素3]
* 可以存放不同类型的元素，可重复，有序，元素可修改

2. 列表的索引：
* 正向：从0开始
* 反向：从-1开始

3. 列表元素的查看，修改，删除：
* 查看：list[0]
* 修改：list[0]='a'
* 删除：del list[0]

4. 列表的切片：
* 语法：s[start`:end:`step]
* 特点：左闭右开，start和step默认为1可省略

5. 列表的常用方法：
|方法|作用|样例|
|:---:|:---:|:---:|
|append()|在列表尾部追加元素|'''s.append(100)'''|
|insert()|在指定索引之前，插入该元素|'''s.insert(0,666)'''|
|remove()|移除列表中匹配到的第一个值|'''s.remove(44)'''|
|pop()|删除列表中指定索引位置的元素（未指定则默认最后一个）|'''s.pop(2)'''|
|sort()|对列表进行排序（元素类型一致才能排序）|'''s.sort()'''|
|reverse()|反转列表元素|'''s.reverse()'''|

6. python中常用于数据统计的语句：
* min(): 获取最小值
* max(): 获取最大值
* sum(): 求和
* len(): 获取元素个数

7. 快速合并两个列表：
* 使用+直接合并
* 使用星号解包（将数据容器揭开成独立元素）

8. 如何判断一个元素是否存在与列表之中：
* 可以使用in运算符，结果返回为布尔值

9. 列表推导式：
* 含义：列表推导式是按照一定规则快速生成一个列表的方法
* 格式: [要插入的数据 for i in 列表 if ]

## **字符串**
1.特点：不可变性（不可修改），有序性，可迭代性

2.索引：
* 正向：从0开始
* 反向：从-1开始

3. 字符串的切片：
* 语法：s[start`:end:`step]
* 特点：同样左闭右开，start默认为0，step默认为1

4. 字符串常用方法;
|方法|作用|样例|
|:---:|:---:|:--:|
|find()|在字符串里找子串，返回第一次出现位置索引，找不到返回-1|s.find('a')|
|count()|统计子串在字符串中出现的次数|s.count('a')|
|upper()|将所有字符串转换为大写|s.upper()|
|lower()|将所有字符串转换为小写|s.lower()|
|split()|将字符串按指定分隔符分割成列表|s.split('a')|
|strip()|将去除字符串两端空白字符或指定字符|s.strip()|
|replace()|将字符串中的指定子串替换为新的子串|s.replace('a','b')|
|startwish()|检查字符串是否以指定子串开头，返回布尔值|s.startwish('a')|
