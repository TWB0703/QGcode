# **数据容器**
## **列表**
1. 列表的定义方式及特点：
* 列表名 = [元素1 ， 元素2 ，元素3]
* 可以存放不同类型的元素，可重复，有序，元素可修改

2. 列表的索引：
* 正向：从0开始
* 反向：从-1开始

3. 列表元素的查看，修改，删除：
* 查看：list[0]
* 修改：list[0]='a'
* 删除：del list[0]

4. 列表的切片：
* 语法：s[start`:end:`step]
* 特点：左闭右开，start和step默认为1可省略

5. 列表的常用方法：
|方法|作用|样例|
|:---:|:---:|:---:|
|append()|在列表尾部追加元素|'''s.append(100)'''|
|insert()|在指定索引之前，插入该元素|'''s.insert(0,666)'''|
|remove()|移除列表中匹配到的第一个值|'''s.remove(44)'''|
|pop()|删除列表中指定索引位置的元素（未指定则默认最后一个）|'''s.pop(2)'''|
|sort()|对列表进行排序（元素类型一致才能排序）|'''s.sort()'''|
|reverse()|反转列表元素|'''s.reverse()'''|

6. python中常用于数据统计的语句：
* min(): 获取最小值
* max(): 获取最大值
* sum(): 求和
* len(): 获取元素个数

7. 快速合并两个列表：
* 使用+直接合并
* 使用星号解包（将数据容器揭开成独立元素）

8. 如何判断一个元素是否存在与列表之中：
* 可以使用in运算符，结果返回为布尔值

9. 列表推导式：
* 含义：列表推导式是按照一定规则快速生成一个列表的方法
* 格式: [要插入的数据 for i in 列表 if ]

## **字符串**
1.特点：不可变性（不可修改），有序性，可迭代性

2.索引：
* 正向：从0开始
* 反向：从-1开始

3. 字符串的切片：
* 语法：s[start`:end:`step]
* 特点：同样左闭右开，start默认为0，step默认为1

4. 字符串常用方法;
|方法|作用|样例|
|:---:|:---:|:--:|
|find()|在字符串里找子串，返回第一次出现位置索引，找不到返回-1|s.find('a')|
|count()|统计子串在字符串中出现的次数|s.count('a')|
|upper()|将所有字符串转换为大写|s.upper()|
|lower()|将所有字符串转换为小写|s.lower()|
|split()|将字符串按指定分隔符分割成列表|s.split('a')|
|strip()|将去除字符串两端空白字符或指定字符|s.strip()|
|replace()|将字符串中的指定子串替换为新的子串|s.replace('a','b')|
|startwish()|检查字符串是否以指定子串开头，返回布尔值|s.startwish('a')|

## **元组**
1. 元组的特点：
* 元素可以重复
* 有序
* 不可修改

2. |查询方法|作用|
   |:---:|:---:|
   |count()|统计指定元素的个数|
   |index()|获取指定元素的下标，不存在则报错|
   
3. 定义单元素元组时，需在结尾上加上逗号，如('A',)

4. 组包与解包：
* 组包：将多个值合并到一个容器中
* 解包“：将容器中的元素分别赋值给多个变量
* 在元组解包时，星号表示收集剩余所有元素，允许我们处理不确定的元素

## **集合**
1. 特点：无序，不可重复，可修改

2. 定义：
* 集合名={元素1，元素2，元素3}
* 空集合：set()

3. 常用操作：
* 添加：s1.add()
* 删除：s1.remove(...)/s1.pop()---随机弹出/s1.pop()
* 交，并，差集：s1.intersection(s2)---"&"/s1.union(s2)---"|"/s1.difference(s2)---"-"

4. 集合推导式：
* 变量名称={i 表达式 for i in 集合 if 条件}

## **字典**
1. 字典的储存形式及特点：
* 字典里存储的是键值对形式数据
* key是唯一的，可修高

2. 字典定义的语法：
* 定义：dict1={"A"：1,"B"=2,"C"=3}
* 获取：v=dict["A"]

3. 注意事项：
* value可以是任意类型，而key必须是不可变类型
* key决不允许重复
* 字典是没有索引下标的

4. 常用方法：
|方法|作用|
|:---:|:---:|
|字典[key]=value|添加/修改|
|del 字典[key]/ value=字典.pop(key)|删除|
|字典[key]/字典.get(key)/字典.values()/字典.items()|查询|
